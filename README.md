## 🌟 **درباره پروژه**

## 🚀 **شروع پروژه**

### **پیش‌نیازها**
```bash
Node.js 18.17+
npm 9.0+ یا yarn 1.22+
Git
```

### **نصب**
```bash
# کلون پروژه
git clone https://github.com/your-username/saeed-pay.git
cd saeed-pay

# نصب وابستگی‌ها
npm install
# یا
yarn install

# کپی فایل متغیرهای محیطی
cp .env.example .env.local

# اجرای پروژه در حالت development
npm run dev
# یا
yarn dev
```

### **متغیرهای محیطی**
```env
# Database
DATABASE_URL="postgresql://..."

# Authentication
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"

# Payment Gateway
PAYMENT_GATEWAY_URL="https://api.payment.com"
PAYMENT_GATEWAY_KEY="your-payment-key"

# AI Services
OPENAI_API_KEY="sk-..."

# SMS Service
SMS_API_KEY="your-sms-key"
SMS_API_URL="https://api.sms.ir"
```
